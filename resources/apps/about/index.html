Slime Client v0.3.0 Beta

Slime Network is a free, open source, federated video game marketplace concept.

To follow its development, or to contribute, see our public repositories on our Github

Slime Network is built on:

  • Tauri
  • React + TypeScript
  • Chia blockchain
  • WalletConnect v2
  • libtorrent
